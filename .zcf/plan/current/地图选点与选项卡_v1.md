## 上下文

- 目标：新增定位地图选点（无 Key、可联网，基于 OSM）与“通用/相机/定位”选项卡分区。
- 约束：同一 APK；无需第三方地图 Key；允许联网加载瓦片。

## 交付物（v1）

- TabLayout + ViewPager2 三分区：通用 / 相机 / 定位
- 地图选点页：WebView + Leaflet（本地资源），点击地图回填经纬度
- 配置保存不互相覆盖（各分区只更新自身字段）

## 验收标准（v1）

- 定位页点击“选择位置”，地图选点后经纬度回填
- 三个 Tab 分区功能正常，保存互不覆盖
- CTF App 显示定位为选择的坐标

## 实施步骤（原子操作）

1. 引入 Material + ViewPager2 依赖，新增 INTERNET 权限。
2. 主界面改为 TabLayout + ViewPager2，接入三个 Fragment。
3. 拆分通用/相机/定位 UI，并实现各自的保存逻辑。
4. 新增 MapPickerActivity + assets（Leaflet + map_picker.html）。
5. 本地构建验证并输出设备验证步骤。

## 风险与边界

- 依赖 OSM 联网瓦片，离线不可用。
